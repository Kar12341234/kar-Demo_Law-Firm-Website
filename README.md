# Law Firm Website Demo

A professional law firm website demo built for a Fiverr / Upwork portfolio. The design targets law firms, consultants, accountants, and other professional services that need a trustworthy, high-end business website.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Responsive design
- Netlify deployment ready

## Scripts

```bash
npm run dev
npm run build
npm run start
```

## Deployment

This project includes `netlify.toml` and `@netlify/plugin-nextjs` for Netlify deployment.
