# Professional Law Firm Website Demo

## Project Overview

This is a professional law firm website demo designed for portfolio presentation on Fiverr, Upwork, Taobao, and personal client pitches. The site presents a trustworthy boutique law firm brand with a clear service structure, restrained visual design, and polished business-focused content.

The demo can also be adapted for consulting firms, accounting practices, advisory companies, and other professional service providers that need a credible online presence.

## Features

- High-end hero section with clear consultation CTA
- Practice area grid for legal and professional services
- About firm section focused on trust, experience, and confidentiality
- Attorney profile cards with roles, specialties, and short bios
- Case results and achievement metrics
- Client testimonials
- Dedicated consultation CTA section
- Contact form with name, email, phone, and case description fields
- Professional footer with office details and navigation
- Responsive layout for desktop, tablet, and mobile

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Lucide React icons
- Netlify-ready build configuration

## Portfolio Purpose

This project is built as a reusable portfolio demo to show potential clients that you can create polished, business-ready websites for:

- Law firms
- Consultants
- Accounting firms
- Financial advisors
- Corporate service providers
- Other professional service companies

The design emphasizes credibility, clear messaging, professional structure, and a premium but understated visual style.

## Netlify Ready

This project includes a `netlify.toml` file with production build settings:

```toml
[build]
  command = "npm run build"
  publish = ".next"
```

To deploy on Netlify:

1. Import the GitHub repository into Netlify.
2. Select the `main` branch.
3. Use `npm run build` as the build command.
4. Use `.next` as the publish directory.
5. Deploy the site.

## Scripts

